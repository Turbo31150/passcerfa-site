Comment ça marche

  1. 1 — Connexion unique : vous vous identifiez une fois sur le site officiel FranceConnect+ (impôts, Ameli, ou identité numérique La Poste).
  2. 2 — Récupération auto : avec votre accord, l'app récupère vos justificatifs auprès des administrations (avis fiscal, quotient CAF, RIB, attestations) via les 14 APIs gouv branchées.
  3. 3 — Pré-remplissage CERFA : tous les formulaires CERFA (MDPH 15692, AAH 13750, 2042, etc.) sont remplis automatiquement. Vous validez champ par champ.
  4. 4 — Soumission directe : envoi direct au site de l'État via API officielle. Vous recevez le reçu localement.
  5. 5 — Purge : à la fermeture de l'onglet, le token et toutes les données disparaissent. Rien n'est conservé.

🔒 Sécurité — zéro stockage client

  • ✅ Aucune base de données utilisateur — pas de comptes, pas de profils stockés
  • ✅ Token FranceConnect+ chiffré en sessionStorage (Web Crypto AES-256-GCM)
  • ✅ Purge automatique à la fermeture d'onglet (beforeunload + visibilitychange)
  • ✅ Edge Functions Netlify stateless — aucun log de payload, uniquement métriques techniques
  • ✅ Anonymisation locale (Ollama gemma3) avant tout appel LLM cloud — NIR/RIB/adresses jamais transmis en clair
  • ✅ LLMs européens prioritaires (Mistral FR) — pas de transfert hors UE par défaut
  • ✅ Pas de cookies tiers, pas de tracker, CSP strict
  • ✅ Code source auditable — repo privé client, audit RGPD/CNIL/AIPD livré
