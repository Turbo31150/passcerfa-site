MDPH — Dossier handicap (CERFA 15692)

Constituez ou renouvelez votre dossier MDPH. Formulaire CERFA 15692 pré-rempli. Adapté aux profils cognitifs et visuels.

📊 410K recherches/mois · Publics : Personnes en situation de handicap cognitif, visuel, moteur — aidants

Commencer — voir les forfaits ← Retour accueil

🔒 Connexion unique via FranceConnect+ · aucune donnée stockée · token purgé à la fermeture

Comment ça se passe — étape par étape

  1. 1 — Identification FranceConnect+ : Connexion via impôts ou Ameli. Données d'identité récupérées automatiquement.
  2. 2 — Pré-remplissage CERFA 15692 : Identité, adresse, coordonnées médicales pré-remplies depuis vos données FC+.
  3. 3 — Compléments médicaux : Vous ajoutez les pièces médicales (certificat médical, justificatifs de soins).
  4. 4 — Envoi à votre MDPH : Soumission directe à la MDPH de votre département. Récépissé avec numéro de suivi.
  5. 5 — Suivi dossier : Notification automatique dès mise à jour de votre dossier.

Accessibilité — modes disponibles

  • ✅ Mode dyslexie (police OpenDyslexic, espacement augmenté, fond beige)
  • ✅ Mode contraste élevé (fond noir, texte blanc, accents jaunes)
  • ✅ Agrandissement texte progressif jusqu'à 24px
  • ✅ Navigation clavier complète (Tab, Escape, Enter)
  • ✅ Compatible lecteur d'écran (NVDA, JAWS, VoiceOver)
  • ✅ Interface FALC disponible via l'assistant IA (Forfait Autonomie+)
  • ✅ Respect prefers-reduced-motion

Accès à cette démarche

Cette démarche est disponible à partir du Forfait Découverte (gratuit) pour les 5 démarches prioritaires, ou incluse dans le Forfait Autonomie à 8,90 €/mois (30 démarches complètes).
